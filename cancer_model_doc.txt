# machine-learn

A practical machine learning repository for daily model experimentation and research.

## Overview

This project explores core machine learning concepts through hands-on implementation with real-world datasets. Primary focus on regression models, hyperparameter optimization, and rigorous evaluation methodology.

## Features

- **Linear Regression** — Fundamental supervised learning approach
- **K-Nearest Neighbors** — Non-parametric regression model
- **Hyperparameter Tuning** — GridSearchCV with cross-validation
- **Pipeline Architecture** — Automated preprocessing and modeling
- **Model Evaluation** — R², RMSE, and comparative analysis
- **Best Practices** — Train-test split, avoiding data leakage

## Quick Start

### Requirements
```
Python 3.8+
scikit-learn
matplotlib
numpy
jupyter
```

### Setup
```bash
# Clone repository
git clone https://github.com/username/machine-learn.git
cd machine-learn

# Create virtual environment
python -m venv .venv
.venv\Scripts\activate

# Install dependencies
pip install scikit-learn matplotlib numpy jupyter
```

## Usage

Open and run notebooks in Jupyter:
```bash
jupyter notebook cancer.ipynb
jupyter notebook line_regression.ipynb
```
